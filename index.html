Stroke Subtype Classification

We are developing an open-source AI tool that helps clinicians distinguish between ischemic and hemorrhagic stroke using only early clinical information—before imaging is available.

The goal is to support decision-making in low-resource settings where CT scans are not accessible, reducing the risk of fatal treatment errors. Built on real-world hospital data and designed for global deployment, this tool aims to improve stroke care where it's needed most.
